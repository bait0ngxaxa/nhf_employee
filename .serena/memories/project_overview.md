Employee NHF is an internal Next.js app for NHF that manages employee data, IT support tickets, notifications, and additional internal modules including stock/material requests. Tech stack: Next.js 15 app router, React 19, TypeScript, Tailwind CSS v4, Radix UI/shadcn-style UI primitives, Prisma ORM with MySQL, NextAuth, Zod, SWR, Vitest, ESLint. Rough structure: `app/` for routes and API handlers, `components/` for dashboard/UI modules, `hooks/`, `lib/` for services/utilities, `constants/`, `types/`, `prisma/` for schema/seed, `__tests__/` for tests. README indicates role-based internal dashboard with employee, IT support, stock, and notification features.
